机器学习基石六

Theory of Generalization

growth function \(m_{H}(N)\):max number of dichotomies


假设我告诉你今天有一个Hypothesis Set，它最小的break point是2，这告诉我们什么的呢？如果我今天只有一个点的时候，那我的成长函数一定能够shatter/击败所有的情形，我能在某种点的组合上产生两种dichotomy

N = 1:every \(m_{H}(N)\) = 2 by definition
N = 2:every \(m_{H}(N)\) < 4 by definition(so maximum possible=3)

要是今天有两个点，那么就不能shatter，因为2是break point，那最多有3种dichotomy

shtter是把n个点的各种可能性都表示出来。break point是最小不能表示全部可能性的点数，dichotomy是点的各种组合，成长函数计算的是N个点下的dichotomy
k = 2,任两个点不能被shatter。上图中，1、2、3种dichotomy时都无法shtter2个点，但是出现的第四个要是刚好是圈叉叉时，观察x2和x3发现把x2和x3被shatter了，所以这四个dichotomy不满足我们的前提条件：break point = 2.

经过一番尝试后，发现4个dichotomy是极限（k=2的情况下）

N = 1:every \(m_{H}(N)\) = 2 by definition
N = 2:every \(m_{H}(N)\) < 4 by definition(so maximum possible=3)
N = 3:maximum possible = 4 \(\ll 2^{3}\)

刚才N=2时最多是3种，现在N=3，我们得到的结论是最多是4种，发现刚才N=2时，3和4好像只差一点点（成长函数和2的N次方），现在N=3时4和8好像差的很多，所以我们发现说：

break point k restricts maximum possible \(m_{H}(N)\) a lot for N > k
idea:\(m_{H}(N) \leq\)maximum possible \(m_{H}(N)\) given k \(\leq\) poly(N)

mH实际上和我们的hypothesis set有关，到底我的mH在有某个break point的情况下，到底能够产生多少种可能性（dichotomy），如果最多最多的这种可能性也是个多项式的话，那么我们的成长函数mH也是个多项式。上次课我们有讲到，要是mH是个多项式，然后如果我们能够将它放进之前的hoeffding不等式里，那么就可以说像perceptron这种情形，learning是做得到的。

bounding function B(N,k): maximum possible \(m_{H}(N)\) when break point = k
可以认为是我今天不管成长函数长什么样，而是可以做出多少种排列组合，相当于把N个点表示成一个向量，然后遮住几个点，只看k个点，我们希望没有\(2^k\)种情况出现（shatter）
combinatorial quantity: maximum number of length-N vectors with \(\color{Blue}{\bullet} \color{Red}{\times} \) while ‘no shatter’ any length=k subvectors
irrelevant of the details of H e.g. B(N,3) bounds both
positive intervals(k = 3)
1D perceptrons(k = 3)

能不能证明：B(N,k) \(\leq\) poly(N)?



Know: B(N,k) = \(2^N\) - 1 for N = k —-removing a single dichotomy satisfies ‘breaking condition’

这样我们就完成了一半了。接下来我们要填剩下的表，剩下的表要困难一些。
Motivation: B(4,3) shall be related to B(3,?) —- ‘adding’ one point from B(3,?)
next: reduce B(4,3) to B(3,?)

我们的上线会被一个上线的上线所约束

成长函数会被上限函数bound，上限会被上限的上限bound，上限的上限会被某个多项式bound，这个多项式和你的break point有关，所以只要你的成长函数有那个beak point，那就不要怕，你最后一定会是一个多项式。

want:$$\mathbb{P}[\exists h \in H s.t.|E_{in}(h) - E_{out}(h)| > \epsilon ]\leq 2 \color{Orange}{m_{H}(N)}\cdot exp(-2 \epsilon^2 N )$$
actually,when N large enough,
$$\mathbb{P}[\exists h \in H s.t.|E_{in}(h) - E_{out}(h)| > \epsilon ]\leq 2\color{Red}{\cdot2} \color{Orange}{m_{H}(\color{Blue}{2}N)}\cdot exp(-2 \cdot \color{Purple}{\frac{1}{16}}\epsilon^2 N )$$
