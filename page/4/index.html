问题描述

给定由n个整数（可能为负数）组成的序列a1,a2,…,an,以及一个正整数m，要求确定次序列的m个不相交子段的总和达到最大。最大子段和问题是最大m子段和问题当m = 1时的特殊情形。

问题分析

设b(i,j)表示数组a的前j项中i个子段和的最大值，且第i个子段含a[j]\(1 \leq i \leq m,i \leq j \leq n\)，则所求的最优值显然为$$\underset{m \leq j \leq n}{max}b(m,j)$$
与最大子段问题相似，计算b(i,j)的递归式为：$$b(i,j)=max\begin{Bmatrix}b(i,j-1) + a[j],\underset{i-1 \leq t < i}{maxb(i-1,t) + a[j]}\end{Bmatrix}  (1 \leq i \leq m,i \leq j \leq n)$$
其中，b(i,j-1) + a[j]表示第i个子段含a[j - 1],a[j]和a[j - 1]都在第i子段里
\(\underset{i - 1 \leq i < j}{max\begin{Bmatrix}b(i - 1,t) + a[j]\end{Bmatrix}}\)项表示第i个子段仅含a[j]。
初始时，b(0,j) = 0,\(1 \leq j \leq n \); b(i,0) = 0,\(1 \leq i \leq m \)。

树状数组

概述

树状数组(Binary Indexed Tree，BIT，二分索引树)，其初衷是解决数据压缩里的累积频率(Cumulative Frequency)的计算问题，现多用于高效计算数列的前缀和。它可以以\(O(\log n)\)的时间得到\(\sum_{i=1}^N\) a[i]，并同样以\(O(\log n)\)对某项加一个常数

例子
